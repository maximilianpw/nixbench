Failure notes

The hard cases were fake-lib and hidden-shape traps.

flake

Metadata shape

Both runs missed required app metadata on the flake task while satisfying the visible structure.

lib

Unavailable helpers

Codex used escapeShellArgs and getExe; Claude used concatStringsSep. The fake evaluator library intentionally rejected them.

license

Wrong plausible values

Both models chose mit for the Python package where the hidden evaluator expected asl20.

shape

Function versus set

Claude's package-stdenv-cli output returned a function where the evaluator expected a package set.
